# Methods

A collection of machine learning & computer vision method notes, utilities, and experimental references.

이 레포는 개인 학습용으로, 다양한 ML/CV 방법론을 구조적으로 정리하고  
필요한 유틸리티 코드(python modules), 실험 예제, 포맷 레퍼런스를 기록하기 위한 공간입니다.

---

## 📦 Included Topics

여기에는 아래와 같은 ML/CV 관련 내용을 포함하게 됩니다:

- Panoptic Segmentation formats (COCO-style)
- 추가 예정

또한 필요 시 submodules처럼 독립된 디렉토리들로  
각 방법론을 분리해 나갈 예정입니다.

---

## 🚀 How to Use

1. 각 방법론별 폴더에 들어가면  
   관련 코드, 샘플 JSON, 노트북, 설명 문서가 있습니다.
2. 파이썬 유틸은 직접 import 해서 사용할 수 있습니다.
3. 새로운 방법론이나 실험을 추가하려면  
   `methods/새폴더명` 형태로 폴더를 만들어 정리합니다.
